Use Case Descriptions

Use Case ID Use Case Name Primary Actor Secondary Actors Description
UC-01 Register Account Visitor Manager New user creates account with email verification. Manager oversees registration process and account approvals.
UC-02 Login Visitor - Existing user authenticates using email/phone and password to access system features.
UC-03 Browse Menu Customer Chef, Manager View available dishes with real-time updates from Chef on availability. Manager controls menu visibility.
UC-04 Query AI Chatbot Customer Manager Get personalized dish recommendations and order assistance. Manager monitors chatbot performance.
UC-05 Rate Order/Delivery Customer Chef, Delivery, Manager Provide ratings for food quality and delivery service. Affects reputation scores monitored by Manager.
UC-06 View Customer/Rider Reputation Customer, Manager Delivery Check reputation scores before accepting orders or deliveries. Cross-verification between actors.
UC-07 Add Funds/Check Balance Customer Manager Manage wallet balance for payments. Manager handles transaction disputes.
UC-08 Use Forum Promotion VIP Customer Manager Access exclusive VIP discounts and promotions managed by system administrator.
UC-09 Search Dishes Customer Chef Search with filters like cuisine, price, dietary restrictions. Chef updates dish metadata.
UC-10 Prepare Dish Chef Customer, Delivery Mark dishes as prepared and ready. Notifies Customer and triggers Delivery assignment.
UC-11 Accept Delivery Bid Delivery Person Customer, Manager Accept delivery through bidding system. Customer sees delivery acceptance, Manager oversees bidding.
UC-12 Accept Order Chef, Delivery Customer Chef accepts cooking request, Delivery accepts delivery. Customer receives notifications.
UC-13 Assign Deliveries Manager Delivery, Customer Manually assign or oversee automatic delivery assignments. Ensures optimal routing.
UC-14 Receive Complaints Manager Customer, Chef, Delivery Handle complaints from customers about food or delivery. Coordinate resolution with relevant parties.
UC-15 Use AI Chatbot Customer, Manager Chef AI-powered support for customers and analytics dashboard for Manager. Chef receives feedback insights.
- Place Order Customer Chef, Delivery, Manager Submit order that triggers Chef preparation, Delivery bidding, and Manager oversight.
- Track Order Status Customer Chef, Delivery Real-time tracking of preparation (Chef) and delivery (Delivery Person) status.
